ls

Команда ls используется для вывода списка файлов и каталогов в текущем рабочем каталоге.

ls -l

cd

Команда cd используется для изменения текущего рабочего каталога.

cd Documents

pwd

Команда pwd выводит полный путь текущего рабочего каталога.

pwd

mkdir

Команда mkdir используется для создания нового каталога.

mkdir new_directory

rm

Команда rm используется для удаления файлов или каталогов.

rm file.txt
rm -r directory - удаление каталога и его содержимого.

cp

Команда cp используется для копирования файлов и каталогов.

cp source.txt destination.txt

mv

Команда mv используется для перемещения или переименования файлов и каталогов.

mv old_name.txt new_name.txt

touch

Команда touch используется для создания пустого файла или обновления временной метки существующего файла.

touch newfile.txt

cat

Команда cat используется для вывода содержимого файла на стандартный вывод.

cat file.txt

echo

Команда echo выводит строку текста на стандартный вывод или в файл.

echo "Hello, World!"
echo "Hello, World!" > file.txt - записывает текст в файл.

ps

Команда ps отображает информацию о текущих процессах.

ps aux

top

Команда top отображает динамический просмотр системных процессов.

top

kill

Команда kill используется для отправки сигнала процессу, обычно для его завершения.

kill 1234 - завершает процесс с PID 1234.

man

Команда man отображает руководство по использованию команды.

man ls

chmod

Команда chmod изменяет права доступа к файлам или каталогам.

chmod 755 script.sh

chown

Команда chown изменяет владельца файла или каталога.

chown user:group file.txt

sudo

Команда sudo позволяет выполнять команды от имени суперпользователя.

sudo apt update

apt

Команда apt используется для управления пакетами в Debian-based системах.

sudo apt install package_name

yum

Команда yum используется для управления пакетами в RHEL-based системах.

sudo yum install package_name

df

Команда df отображает информацию о свободном дисковом пространстве.

df -h

du

Команда du оценивает использование дискового пространства.

du -sh

tar

Команда tar используется для архивирования файлов.

tar -czvf archive.tar.gz directory

gzip

Команда gzip используется для сжатия файлов.

gzip file.txt

gunzip

Команда gunzip используется для распаковки файлов, сжатых с помощью gzip.

gunzip file.txt.gz

ping

Команда ping используется для проверки сетевого подключения до узла.

ping example.com

ifconfig

Команда ifconfig используется для настройки сетевого интерфейса.

ifconfig

ssh

Команда ssh используется для подключения к удалённому серверу по протоколу SSH.

ssh user@host

scp

Команда scp используется для безопасной передачи файлов между хостами.

scp file.txt user@remote_host:/path/to/destination

crontab

Команда crontab используется для установки расписания выполнения задач.

crontab -e
