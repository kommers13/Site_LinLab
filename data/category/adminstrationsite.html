Утилиты для администрирования

OpenSSH

Бесплатная реализация протокола SSH, обеспечивающая защищенное удаленное управление и обмен данными между компьютерами.

Подробнее

Ansible

Автоматизационный инструмент с открытым исходным кодом для управления конфигурацией, развертываниями и оркестрации в среде Linux и других операционных системах.

Подробнее

Nagios

Система мониторинга с открытым исходным кодом, предназначенная для отслеживания работы компьютерных систем и сетевых служб.

Подробнее

Puppet

Инструмент автоматизации с открытым исходным кодом, который позволяет управлять конфигурациями и автоматизировать процессы развертывания и управления программным обеспечением.

Подробнее

Cockpit

Простой веб-интерфейс для управления системами Linux через веб-браузер, который позволяет администраторам управлять различными аспектами системы.

Подробнее

Webmin

Бесплатная веб-интерфейсная система управления Unix-подобными системами, которая позволяет администраторам управлять различными аспектами системы через веб-браузер.

Подробнее

Nmap

Бесплатный инструмент сканирования с открытым исходным кодом, используемый для обнаружения сетевых узлов и определения открытых портов на удаленных системах.

Подробнее

Htop

Интерактивный монитор ресурсов с открытым исходным кодом для систем Linux, который предоставляет пользователю детальную информацию о процессах и использовании ресурсов системы.
