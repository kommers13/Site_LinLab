Мультимедийные приложения

GIMP

Это программный пакет, который умеет делать многое из того, что делает Adobe Photoshop, но при этом гораздо дешевле его - он бесплатный.

Подробнее

KDEnlive

Видеоредактор нелинейного типа для ПК, решающий задачи обработки видео до профессионального уровня. Взаимодействует с любыми форматами аудио/видео, поддерживает пропорции соотношения сторон 4 : 3 / 16 : 9 для разнообразных стандартов HD, включая AVCHD/ HDV.

Подробнее

Blender

Профессиональное свободное и открытое программное обеспечение для создания трёхмерной компьютерной графики, включающее в себя средства моделирования, скульптинга, анимации, симуляции, рендеринга, постобработки и монтажа видео со звуком, компоновки с помощью «узлов», а также создания 2D-анимаций.

Подробнее

Inkscape

Свободно распространяемый векторный графический редактор, удобен для создания как художественных, так и технических иллюстраций. Это стало возможным во многом благодаря открытому формату SVG, развиваемому консорциумом W3C.

Подробнее

Digikam

Программа для работы с цифровыми фотографиями, каталогизатор и редактор для Linux, macOS и Windows. Программа DigiKam органично интегрирована в среду рабочего стола KDE.

Подробнее

Gwenview

Программа для просмотра изображений для оконной системы KDE. Текущий сопровождающий проекта — Орельен Гато. Слово «Gwen» означает «белый» на бретонском языке и широко используется в качестве имени. Программа доступна как для KDE 3, KDE 4, так и для KDE 5.
